⚡ Connetti YouTube Data API — aggiornamento automatico
1. Vai su console.cloud.google.com → crea progetto → Abilita YouTube Data API v3 → Crea API Key.
2. Il Channel ID si trova in YouTube Studio → Impostazioni → Info canale → ID canale.
Inserisci i due valori e premi Fetch. I dati si aggiornano ogni volta che apri la pagina.
